Aplicação Angular

Este é um projeto de processo seletivo
para o cargo de estagiário em desenvolvimento.
Esta aplicação possui:

  • Front-end em Angular
  • Back-end em Typescript
  • Telas de login e registro de usuários
  • Rotas

Os próximos desafios são:

  • Comunicação com banco de dados
  • Funcionalidades CRUD de usuários
  • Autenticação de login
  • Cadastrar novos usuários

O maior desafio encontrado foi a implementação
do back-end e a comunicação com o banco de dados.
Com o desenvolvimento desta aplicação, foi
possível observar que a implementação adequada
do back-end antes do front-end permitiria um
resultado mais adequado.
